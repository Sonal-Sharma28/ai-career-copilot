# AI Career Copilot

A full-stack application to help users manage their resumes, find matching jobs, and use AI to generate optimized resumes and cover letters.

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- AI: OpenAI

## Setup Instructions

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` file with `PORT`, `MONGO_URI`, `JWT_SECRET`, and `OPENAI_API_KEY`.
4. Run `npm run dev` to start the server.

### Frontend
1. `cd frontend`
2. `npm install`
3. Run `npm run dev` to start the client.
